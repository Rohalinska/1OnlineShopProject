# Food Delivery Database Project

## Опис
Проєкт реляційної бази даних для сервісу доставки їжі.
Розроблено в межах курсової роботи з дисципліни "Бази даних".

## Технології
- MySQL
- PHP
- Apache (XAMPP)
- HTML / CSS

## Функціонал
- Управління клієнтами
- Управління меню
- Оформлення замовлень
- Облік доставок
- Обробка платежів
- Адміністративна панель

## Структура БД
- Clients
- Orders
- Order_Items
- Menu
- Couriers
- Payments
- Restaurants

## Запуск
1. Запустити XAMPP
2. Імпортувати SQL-файл
3. Відкрити проєкт у браузері через localhost
