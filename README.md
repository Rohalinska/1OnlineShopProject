# Онлайн-магазин

## Опис
Система для перегляду товарів, оформлення замовлень та оплати онлайн.

## Функції
- Реєстрація та авторизація
- Перегляд та пошук товарів
- Оформлення замовлень
- Онлайн-оплата
- Управління товарами для адмінів

## Технології
- MySQL
- Python (Flask або Django)
- HTML, CSS, JS

## Інструкція з запуску
1. Встановити MySQL
2. Створити базу даних та таблиці (schema.sql)
3. Встановити Python та бібліотеку: `pip install -r requirements.txt`
4. Запустити `db_connect.py` або сервер
5. Відкрити у браузері: `http://localhost`
